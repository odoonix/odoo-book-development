کنترل نسخه
===============

**Git** یک سیستم کنترل نسخه توزیع شده است که توسط Linus Torvalds، خالق Linux، در سال 2005 توسعه یافته است. این ابزار به توسعه‌دهندگان امکان می‌دهد کدهای خود را مدیریت کنند و تغییرات را پیگیری کنند.

از مهم‌ترین ویژگی‌های نرم‌افزار گیت می‌تواند به موارد زیر اشاره کرد.

1. **کنترل نسخه توزیع شده**: این بدان معنی است که هر توسعه‌دهنده یک کپی کامل از کل مخزن کد را دارد، که این امر کار بر روی نسخه‌های متعدد، حتی بدون اتصال به شبکه، را ممکن می‌سازد.
2. **شتاب و کارایی بالا**: Git به گونه‌ای طراحی شده است که بسیار سریع باشد و قابلیت‌های بالایی را در پردازش داده‌ها ارائه دهد.
3. **ردیابی دقیق تغییرات**: توسعه‌دهندگان می‌توانند تغییرات مورد نظر خود را در کد پیگیری کرده و به راحتی به نسخه‌های قبلی بازگردند.
4. **شاخه‌سازی و ادغام آسان**: شاخه‌سازی به تیم‌های توسعه اجازه می‌دهد تا روی ویژگی‌های مستقل یا اصلاحات کد به صورت موازی کار کنند و پس از اتمام، آسان ادغام شوند.

کاربردهای مهم این نرم‌‌افزار عبارتند از:

- **توسعه نرم‌افزار**: از آغاز تا انتشار نهایی، Git ابزار اصلی بسیاری از تیم‌های توسعه نرم‌افزار برای پیگیری تغییرات و مدیریت کد است.
- **مدیریت پروژه‌های بزرگ و توزیع شده**: با کمک به تیم‌ها در مکان‌های مختلف برای همکاری، Git پروژه‌های بزرگ و تیم‌های توزیع شده را ممکن می‌سازد.
- **شخصی‌سازی و باز‌نویسی سیستم‌ها**: Git به کاربران این امکان را می‌دهد که سیستم‌ها را به صورت محلی و بدون نیاز به مجوز تغییر دهند و بعداً تغییرات را با بقیه تیم به اشتراک بگذارند.

Git یکی از ابزارهای حیاتی در دنیای توسعه نرم‌افزار امروزی است و بسیاری از پلتفرم‌های معروف مانند GitHub، GitLab و Bitbucket از آن استفاده می‌کنند.

نصب و راه اندازی
----------------------------------

نصب Git روی Ubuntu 24.04 ساده و سریع است. می‌توانید مراحل زیر را دنبال کنید:

1. **به‌روزرسانی سیستم**:
   برای اطمینان از به‌روز بودن سیستم، دستور زیر را در ترمینال اجرا کنید:

.. code-block:: bash

   sudo apt update
   sudo apt upgrade -y


2. **نصب Git**:
   پس از به‌روزرسانی، می‌توانید Git را با استفاده از مدیر بسته APT نصب کنید:

.. code-block:: bash

   sudo apt install git -y


3. **تأیید نصب**:
   برای تأیید نصب موفق Git، دستور زیر را اجرا کنید و نسخه نصب شده را بررسی کنید:

.. code-block:: bash

   git --version

4. **پیکربندی اولیه Git**:
   پس از نصب، باید نام کاربری و ایمیل خود را برای Git پیکربندی کنید:

.. code-block:: bash

   git config --global user.name "نام کاربری شما"
   git config --global user.email "ایمیل شما"


این مراحل به شما کمک می‌کند تا به راحتی Git را روی سیستم خود نصب و پیکربندی کنید. 


دستورات پایه گیت
-----------------------

مخازن ایجاد کنید
++++++++++++++++++++++++

یک مخزن محلی ایجاد کنید -

.. code-block:: bash

    git init


اضافه کردن
++++++++++++++++++++++++

افزودن فایل به مرحله بندی -

.. code-block:: bash

    git add


کامیت کردن تغییرات 
++++++++++++++++++++++++

تمام فایل‌های مرحله‌ای را به git متعهد کنید -

.. code-block:: bash

    git commit

لیست فایل های جدید یا اصلاح شده که هنوز متعهد نشده اند -

وضعیت git
++++++++++++++++++++++++


.. code-block:: bash

    git status

ادغام تمام commit ها در یک commit جدید

git ادغام
++++++++++++++++++++++++

یک شاخه را بر روی شاخه دیگر قرار دهید-

.. code-block:: bash

    git mrege

همگام سازی مخازن
++++++++++++++++++++++++

بارگذاری محتوای محلی در مخزن راه دور -

.. code-block:: bash

    git push

کپی مخزن راه دور شعبه فعلی را واکشی کنید -

.. code-block:: bash

    git pull

مدیریت مخزن‌های 
++++++++++++++++++++++++

.. code-block:: bash

    git add origin
