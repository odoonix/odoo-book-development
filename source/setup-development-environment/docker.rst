نصب داکر
=================

داگر (Docker) یک پلتفرم متن‌باز است که امکان ایجاد، استقرار و مدیریت برنامه‌ها را با استفاده از کانتینرها فراهم می‌کند. کانتینرها به توسعه‌دهندگان این امکان را می‌دهند تا برنامه‌ها و تمامی وابستگی‌های آن‌ها را در یک بسته جداگانه اجرا کنند. با استفاده از داکر، برنامه‌ها می‌توانند به سرعت و به سادگی در محیط‌های مختلف مستقر شده و بدون تداخل با سایر برنامه‌ها اجرا شوند. داکر به ویژه در محیط‌های توسعه و تولید مفید است، زیرا به ایجاد محیط‌های یکسان و تکرارشونده کمک می‌کند و باعث می‌شود تا تغییرات به صورت منزوی و کنترل شده اعمال شوند. این امر به توسعه‌دهندگان و تیم‌های IT کمک می‌کند تا از پایداری و قابلیت پیش‌بینی برنامه‌هایشان اطمینان حاصل کنند.

در ادامه مراحل نصب را گام به گام توضیح خواهیم داد.

مرحله ۱: به‌روزرسانی مخازن سیستم
----------------------------------------

ابتدا، مطمئن شوید که سیستم شما به‌روز است. ترمینال را باز کرده و دستورات زیر را اجرا کنید:

.. code-block:: bash

    sudo apt update
    sudo apt upgrade -y


## مرحله ۲: نصب بسته‌های ضروری

برخی از بسته‌های ضروری برای نصب Docker مورد نیاز هستند. دستورات زیر را اجرا کنید تا این بسته‌ها نصب شوند:

.. code-block:: bash

    sudo apt install \
        ca-certificates \
        curl \
        gnupg \
        lsb-release


مرحله ۳: اضافه کردن کلید GPG رسمی Docker
---------------------------------------------

برای اطمینان از صحت بسته‌های دانلود شده، کلید GPG رسمی Docker را به سیستم اضافه کنید:

.. code-block:: bash

    sudo mkdir -p /etc/apt/keyrings
    curl -fsSL https://download.docker.com/linux/ubuntu/gpg | \
      sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg


مرحله ۴: راه‌اندازی مخزن Docker
----------------------------------------

مخزن Docker را به منابع APT خود اضافه کنید:

.. code-block:: bash

    echo \
    "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] \
    https://download.docker.com/linux/ubuntu \
    $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null


مرحله ۵: به‌روزرسانی مخازن APT
----------------------------------------

پس از اضافه کردن مخزن Docker، مخازن APT را به‌روز کنید:

.. code-block:: bash

    sudo apt update

اگر با خطایی مربوط به دسترسی به `/etc/apt/keyrings/docker.gpg` مواجه شدید، دستورات زیر را اجرا کنید:

.. code-block:: bash

    sudo chmod a+r /etc/apt/keyrings/docker.gpg
    sudo apt update


مرحله ۶: نصب Docker Engine، Docker CLI و Containerd
-----------------------------------------------------

اکنون آماده نصب Docker هستید:

.. code-block:: bash

    sudo apt install \
        docker-ce \
        docker-ce-cli \
        containerd.io \
        docker-buildx-plugin \
        docker-compose-plugin


مرحله ۷: بررسی نصب Docker
----------------------------------------

برای اطمینان از نصب صحیح، وضعیت سرویس Docker را بررسی کنید:

.. code-block:: bash

    sudo systemctl status docker


خروجی باید نشان دهد که Docker فعال و در حال اجرا است.

مرحله ۸: اجرای Docker بدون نیاز به `sudo` (اختیاری)
-------------------------------------------------------

برای اجرای دستورات Docker بدون نیاز به دسترسی ریشه، کاربر خود را به گروه `docker` اضافه کنید:

.. code-block:: bash

    sudo usermod -aG docker $USER


برای اعمال تغییرات، از سیستم خارج شده و مجدداً وارد شوید یا از دستور زیر استفاده کنید:

.. code-block:: bash

    newgrp docker


مرحله ۹: تست نصب Docker
----------------------------------------

یک کانتینر تستی را اجرا کنید تا مطمئن شوید Docker به درستی کار می‌کند:


.. code-block:: bash
    
    docker run hello-world



اگر نصب موفقیت‌آمیز باشد، پیامی مبنی بر اجرای صحیح کانتینر خواهید دید.


نکات تکمیلی
----------------------------------------

- **به‌روزرسانی‌های امنیتی:** مطمئن شوید که Docker را به‌صورت منظم به‌روزرسانی می‌کنید تا آخرین به‌روزرسانی‌ها و وصله‌های امنیتی را دریافت کنید.

- **مستندات رسمی:** برای اطلاعات بیشتر و گزینه‌های پیشرفته، به `مستندات رسمی Docker برای اوبونتو <https://docs.docker.com/engine/install/ubuntu/>`_ مراجعه کنید.


با انجام این مراحل، Docker به‌طور کامل روی سیستم اوبونتو 22.04 شما نصب شده و آماده استفاده است. در ادامه، می‌توانید از Docker برای راه‌اندازی محیط‌های توسعه اودوو و مدیریت کانتینرها بهره ببرید.
