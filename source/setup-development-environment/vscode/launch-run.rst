اجرا و رفع خطا
=====================

یکی از قابلیت‌های ارزشمند **Visual Studio Code (VS Code)** برای توسعه‌دهندگان اودوو، امکان اجرای مستقیم نرم‌افزار از طریق تنظیمات پیکربندی شده در فایل **Workspace Configuration** است. در این بخش، به شما نشان می‌دهیم که چگونه می‌توانید با استفاده از بخش **Launch** در VS Code، اودوو را به‌راحتی اجرا کنید و فرآیند توسعه خود را تسریع ببخشید.


در فایل تنظیمات Workspace پروژه شما، یک نمونه اجرایی (Launch Configuration) به بخش **Launch** اضافه شده است که به‌طور خاص برای اجرای اودوو طراحی شده است. این پیکربندی شامل دستورات و پارامترهای لازم برای راه‌اندازی سرور اودوو به‌صورت محلی است. با این تنظیم، نیازی به اجرای دستی اودوو از طریق خط فرمان نیست و می‌توانید همه‌چیز را از داخل VS Code مدیریت کنید.


برای استفاده از این قابلیت، مراحل زیر را دنبال کنید:

1. **باز کردن بخش Run and Debug:**
   ابتدا به نوار کناری VS Code بروید و روی آیکون **Run and Debug** (معمولاً به شکل یک مثلث با یک باگ در کنار آن) کلیک کنید. این بخش به شما امکان دسترسی به تنظیمات اجرایی پروژه را می‌دهد.

2. **انتخاب نام "Odoo Run":**
   در منوی کشویی که در بالای پنجره Run and Debug ظاهر می‌شود، گزینه‌ای به نام **"Odoo Run"** را خواهید دید. این نام مربوط به نمونه اجرایی است که قبلاً در تنظیمات Launch تعریف شده است. آن را انتخاب کنید.

3. **اجرای اودوو با کلید F5:**
   پس از انتخاب "Odoo Run"، کافی است کلید **F5** را فشار دهید. این کار سرور اودوو را با تنظیمات مشخص‌شده راه‌اندازی می‌کند و شما می‌توانید خروجی آن را مستقیماً در ترمینال داخلی VS Code مشاهده کنید یا از ابزارهای دیباگ برای رفع اشکال استفاده کنید.


این ویژگی در کنار ابزار **Otoolbox** که در بخش قبلی معرفی شد، قدرت بیشتری پیدا می‌کند. در حالی که Otoolbox محیط توسعه را آماده و به‌روز نگه می‌دارد، تنظیمات Launch در VS Code به شما اجازه می‌دهد اجرای اودوو را با یک کلیک یا میانبر ساده آغاز کنید. ترکیب این دو ابزار، تجربه‌ای یکپارچه و کارآمد را برای توسعه‌دهندگان اودوو فراهم می‌کند.

به عنوان مثال، پس از اجرای دستور `otoolbox dev start` برای آماده‌سازی محیط و باز کردن VS Code، می‌توانید بلافاصله به بخش Run and Debug بروید و با انتخاب "Odoo Run" و زدن F5، سرور اودوو را فعال کنید. این جریان کاری نه‌تنها زمان را صرفه‌جویی می‌کند، بلکه به شما امکان می‌دهد به‌سرعت بین کدنویسی، تست و دیباگ کردن جابه‌جا شوید.


اضافه کردن نمونه اجرایی "Odoo Run" به تنظیمات Launch در VS Code، یکی از راه‌های مؤثر برای ساده‌سازی فرآیند توسعه اودوو است. با دنبال کردن مراحل ساده بالا، می‌توانید اودوو را مستقیماً از محیط ویرایشگر خود اجرا کنید و با استفاده از ابزارهایی مثل Otoolbox، بهره‌وری خود را به سطح بالاتری برسانید. این ترکیب، توسعه و شخصی‌سازی اودوو را به تجربه‌ای روان‌تر و حرفه‌ای‌تر تبدیل می‌کند.
