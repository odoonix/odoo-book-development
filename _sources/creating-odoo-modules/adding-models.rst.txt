افزودن مدل‌ها
=============

در Odoo مدل‌ها کلاس‌های پایتون هستند که ساختار داده‌ای و منطق مربوط به یک موجودیت کسب‌وکاری را تعریف می‌کنند. هر مدل معمولاً به یک جدول پایگاه‌داده نگاشت می‌شود و فیلدها ستون‌های آن هستند.

انواع مدل‌ها:

- مدل‌های معمولی (پایدار): ذخیره در دیتابیس.
- مدل‌های انتزاعی (_abstract): فقط برای ارث‌بری و اشتراک منطق بین مدل‌ها.
- مدل‌های موقت (TransientModel): برای ویزاردها و داده‌های موقتی.

مثال ساده مدل:

.. code-block:: python

   from odoo import models, fields


   class MyModel(models.Model):
       _name = 'my.model'
       _description = 'My Custom Model'

       name = fields.Char(string="Name")
       active = fields.Boolean(default=True)


روند اضافه کردن مدل جدید:

1. فایل پایتون جدیدی در پوشه `models/` ایجاد کنید (مثلاً `visa_application.py`).
2. در `__init__.py` ماژول، `from . import models` یا `from .models import visa_application` را اضافه کنید تا مدل بارگذاری شود.
3. ماژول را ارتقا دهید (از منو یا دستور CLI) تا ساختار دیتابیس ساخته شود.

نمونهٔ ساده ایجاد مدل و بررسی آن در حالت توسعه: فعال کردن حالت توسعه -> تنظیمات عمومی -> فنی -> ساختار دیتابیس -> Models
